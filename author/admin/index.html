មាន សូនីតា ទាំងស្អាតៗ ទាំងពូកែបែបនេះ បុរសណាមិនលង់ស្នេហ៍

By admin on Sunday, April 22, 2018

តារាសម្តែងកញ្ញា មាន សូនីតា កំពុងតែមានឈ្មោះល្បីល្បាញ និងទទួលប្រជាប្រិយភាពជាខ្លាំងពីសំណាក់មហាជន។ ដោយតារាស្រីរូបនេះមិនត្រឹមតែមានទេពកោសល្យខាងសិល្បៈនោះទេ ក៏វិវឌ្ឍន៍ខ្លួនយ៉ាងជោគជ័យក្លាយជាថៅកែស្រីទាំងវ័យក្មេង ដែលមានហាងលក់សម្លៀកបំពាក់ច្នៃពីក្រម៉ា មានឈ្មោះថា ផ្ទះស្រលាញ់ និងឧស្សាហ៍នាំចេញទៅក្រៅប្រទេសក្នុងពិធីតាំងពិពណ៌ផ្សេងៗទៀតផង។ ជាមួយនឹងភាពជោគជ័យទាំងការងារ និងបញ្ចប់ការសិក្សាថ្នាក់ខ្ពស់ បែបនេះ គេក៏សម្គាល់ថា មាន សូនីតា ចូលចិត្តធ្វើដំណើរកម្សាន្តចេញទៅក្រៅប្រទេសដើម្បីលំហែកាយផងដែរ។ យ៉ាងណាមិញថ្មីៗនេះកញ្ញា ក៏បានចេញទៅប្រទេសឡាវ និងបានបង្ហោះរូបភាពបង្ហាញនូវភាពរីករាយនៃដំណើរកម្សាន្ត រួមទាំងមានភាពស្រស់ស្អាត និង សិចស៊ី ធ្វើឲ្យអ្នកលេងបណ្តាញសង្គមក៏ដូចជាអ្នកគាំទ្រកោតសរសើរមិនដាច់ពីមាត់៕

ស្អាតបែបនេះតិចច្រឡំថាប្រទេសគេ តំបន់ធម្មជាតិ “ភ្នំពស់រលួសធំ” កំពុងលាក់ខ្លួនសែនឆ្ងាយ ក្បែរព្រំប្រទល់ខ្មែរ-ថៃ

By admin on Sunday, April 22, 2018

ឧត្តរមានជ័យ៖ នៅមានមនុស្សតិចណាស់ដែលស្គាល់តំបន់ទេសចរណ៍ធម្មជាតិមួយនេះ ភ្នំពស់​ រលួសធំ​ ដែលជាទីតាំងធម្មជាតិមួយ​ នៅតាមខែ្សក្រវ៉ាត​ នៃបន្ទាត់ព្រំដែននៅខេត្តឧត្តរមានជ័យ​ ដែលមាន​ ដីច្រោះ មានផ្ទៃដីជាង​ ៣០០​ ហិកតា ស្ថិតនៅជាប់ភូមិចំនួនបី  គឺភូមិ​ខ្នាចឬស្សី​ ភូមិ ​ពោធិវង្សសង្កាត់បន្សាយរាក់ និង​ភូមិ​ ត្រពាំងទូង​ ក្នុង​សង្កាត់​កូន​គ្រៀល​ ក្រុងសំរោង ខេត្តឧត្តរមានជយ័។“រមណីយដ្ឋានធម្មជាតិរលួសធំ” មើលទៅហាក់ដូចមានរាងដំបូក ឬដូចសំណង់ប្រាសាទបុរាណដែលមានអាយុកាលយូរឆ្នាំ។ ប៉ុន្តែមូលហេតុដែលនាំឲ្យនៅទីនោះមានរូបរាងបែបនេះ ដោយសារតែការជីកដីយកមកធ្វើផ្លូវ ដូច្នេះហើយមុខជីករបស់អេស្កាវេទ័រ   ពេលភ្លៀងមកមិនរលុះដីទេ  ហេតុនេះហើយទើបមើលទៅដូចមានរាងដំបូកដូច្នេះ។ យ៉ាងណាមិញ យោងតាម រដ្ឋមន្ត្រី​ក្រសួង​ព័ត៌មាន លោក ខៀវ កាញារីទ្ធ បានឲ្យដឹងថា កាលពីព្រឹកថ្ងៃទី១៩ ខែមេសា ឆ្នាំ២០១៨  អភិបាលនៃគណៈអភិបាលខេត្តឧត្តរមានជយ័ បានចុះពិនិត្យទីតាំងតំបន់ ភ្នំពស់​ រលួសធំ នេះ និងគ្រោងរៀបចំអភិវឌ្ឍន៍តំបន់នេះអោយក្លាយជាតំបន់ទេសចរណ៍នាពេលខាងមុខដើម្បីទាក់ទាញភ្ញៀវជាតិនិងភ្ញៀវអន្តរជាតិមកដើរលេងកំសាន្តថែមទៀតផង៕

ពិធីការិនី ណាក់ ស្រីណា៖ “ខំផ្សព្វផ្សាយសម្លៀកបុរាណខ្មែរច្រើនឆ្នាំមកហើយ ប៉ុន្តែមិនជ្រាបក្នុងបេះដូងរបស់ខ្មែរ បែរជាចាញ់ឥទ្ធិពលរបស់គេ”

By admin on Sunday, April 22, 2018

រឿងរ៉ាវសម្លៀកបំពាក់បុរាណកំពុងក្លាយជាប្រធានបទដ៏ក្តៅគគុក ពិភាក្សាវែកញែកគ្នា ពេញបណ្តាញសង្គម។ ដោយមានការលើកឡើងថាកូនខ្មែរជំនាន់ក្រោយពុំទាន់បានយល់ និងមិនដឹងច្បាស់ពីសណ្ឋាននៃការស្លៀកពាក់ ដោយមានការយល់ច្រឡំជាខ្លាំង និង ពុំយល់ដឹងថា សណ្ឋានបែបណាជារបស់ខ្មែរ ឬរបស់ថៃ។ មិនត្រឹមតែប៉ុណ្ណោះខណៈពេលដែលសិល្បករ សិល្បការិនីខ្មែរ បានបំផុសបំផុលនាំគ្នាស្លៀកពាក់បុរាណបែបនេះ ក៏មានការរិះគន់ និងស្តីបន្ទោសថាទាល់តែភាពយន្តបុរាណថៃផ្ទុះទើបបានយកមកស្លៀក។ យ៉ាងណាមិញពិធីការិនី ណាក់ ស្រីណា ក៏បានលើកឡើងថា ការស្លៀកពាក់បុរាណបែបនេះគឺសិល្បករ សិល្បការិនីខ្មែរជាច្រើនបានបង្ហាញតាមរយៈបទចម្រៀង និងខ្សែភាពយន្តផ្សព្វផ្សាយច្រើនឆ្នាំណាស់មកហើយ។ ប៉ុន្តែគ្មានអ្នកណាចាប់អារម្មណ៍ ស្រាប់តែនៅពេលរឿងថៃបុរាណផ្ទុះ មានប្រជាជន យុវជន យុវតីខ្មែរជាច្រើន នាំគ្នាចាប់អារម្មណ៍។ ដោយកញ្ញា បានសរសេរនៅលើគណនីហ្វេសប៊ុករបស់ខ្លួនថា៖ “ព្រោះ​តែ ខ្វះជាតិសាសន៍និយមទើបទៅជាបែបនេះ? នេះ​មិន​មែន​ជា​ការ​ឆ្លៀតឱកាស​ចង់ល្បីល្បាញអីនោះទេ តែខ្ញុំគ្រាន់តែចង់បង្ហាញពីការឈឺចាប់ចំពោះខ្មែរមួយចំនួនដែលតែងតែឲ្យតម្លៃជាតិសាសន៍ដទៃជាងជាតិសាសន៍របស់ខ្លួន ។ទំរង់សម្លៀកបំពាក់បែបនេះពួកយើងបានប្រើប្រាស់និងព្យាយាមផ្សព្វផ្សាយច្រើនឆ្នាំមកហើយ តែហាក់បីដូចជាមិនសូវបានជ្រៀបចូលក្នុងបេះដូងរបស់ខ្មែរសោះ បែរជានៅតែចាញ់ឥទ្ធិពលរបស់បរទេសដដែល។រូបភាពជាង២ឆ្នាំមុន”៕
