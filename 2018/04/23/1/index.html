មុនចាកចេញពីសិល្បៈ ឌីជេក្តិប បានត្រឹមបន្សល់ទុកបទចម្រៀងមួយបទចុងក្រោយជាចំណងដៃឲ្យអ្នកគាំទ្រ (មានវិដេអូ)

By admin on Monday, April 23, 2018

ឌីជេ ក្តិប ជាតារាមួយរូបដែកប្រឡូកក្នុងវិស័យសិល្បៈជាយូរណាស់មកហើយ ថែមទាំងទទួលបានការគាំទ្រខ្លាំងទៀតផង។ ប៉ុន្តែអ្វីដែលគួរឲ្យសោកស្តាយគឺ តារាប្រុសរូបនេះ បែរជាប្រកាសចាកចេញពីសិល្បៈ បន្ទាប់ពីជួបរឿងឈឺចិត្តមិនអាចរកប្រាក់ព្យាបាលឪពុកពីជម្ងឺបាន។ ជុំវិញការប្រកាសនេះ បានធ្វើឲ្យមហាជន​ និងអ្នកគាំទ្រជាច្រើនសម្ដែងការសោកស្តាយនឹងស្នាដៃល្អៗ របស់លោក ដែលបានបន្សល់ទុក។

ជាក់ស្ដែង មុននឹងចាកចេញ ឌីជេក្ដិប ក៏បានទុកនូវចំណងដៃចុងក្រោយជូនគ្រប់គ្នាផងដែរ គឺបទ I’m Ok ។ តោះស្តាប់បទចុងក្រោយរបស់លោកទាំងអស់គ្នា៖
